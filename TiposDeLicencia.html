Tipos de licencias


OEM (Original Equipment Manufacturer -Fabricante Original de Equipo):


Este tipo de licencia obliga que la venta del software sea adquirido con un equipo nuevo, la venta de este software queda prohibido si no se ajusta a esta condición. Normalmente estas licencias se ven aplicadas a los sistemas operativos y pueden imponer limitaciones al uso de este, por ejemplo la cantidad máxima veces que el usuario puede reinstalar el software.

Licencia retail:


Esta licencia otorga la completa propiedad del software al usuario, ya sea para venderlo ó cederlo pero no para realizar copias de éste.

Licencias por volumen:


Destinado para usuarios grandes como las empresas, normalmente utiliza condiciones similares a las licencias tipo OEM, con la diferencia que estas no están sujetas a ser adquiridas con equipos nuevos. Primordialmente se estipula una determinada cantidad de equipos para la utilización del software, donde el fabricante tiene la autorización de hacer las comprobaciones correspondientes para que las licencias que se vayan a utilizar sean adquiridas.
